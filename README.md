##  Technology Stack

Backend: Laravel (latest stable preferred)
Frontend: Vue.js (latest stable preferred, Composition API encouraged)
Database: MySQL or PostgreSQL
Real-time: Pusher via Laravel Broadcasting.

### Setup

1. Clone the repo
2. CD exchange-engine and run follwing command
3. >> composer install 
4. >> php artisan migrate:fresh --seed  
5. >> npm install
6. >> Copy env.example to .env
7. >> Run php artisan key:generate
8. Run API with >> php artisan dev
9. Run Frontend with >> npm run dev
10. visit -> http://127.0.0.1:8000
